# Noti-Blog

Este repositorio contiene la aplicación Astro para un blog de noticias, con integración a Sanity para la administración de artículos. La aplicación está diseñada para ser moderna, simple y responsiva, compatible con navegadores de escritorio y móviles. Incluye pruebas básicas con Vitest.

Para iniciar el proyecto localmente, instala las dependencias y ejecuta el servidor de desarrollo:

```
npm install
npm run dev
```

Consulta el archivo `.env.example` para configurar las variables de entorno necesarias.
